Why Nemotron?

Purpose-built for coding, math, scientific reasoning, tool calling, and visual reasoning.

Open Models

Fully transparent training data, techniques, and weights for community innovation.

Compute Efficiency

Model pruning and optimization enabling higher throughput via TensorRT-LLM.

High Accuracy

Built on frontier open models with human-aligned reasoning for agentic workflows.

Flexible Deployment

Deploy anywhere: edge, single GPU, or data center with NIM microservices.

Model Family

Three tiers spanning edge to datacenter, plus a natively multimodal variant.

Nemotron 3 Ultra

NVIDIA's largest Nemotron 3 model — hybrid Mamba-Attention LatentMoE with MTP and 1M context for datacenter-scale agentic reasoning.

550B total / 55B active · 20T tokens · 1M ctx
Pretrain → SFT → RLVR → MOPD

Nemotron 3 Super

Frontier Hybrid Mamba Latent MoE Transformer for reasoning, coding, and agentic tasks on a single GPU.

120.6B total / 12.7B active · 21 RL reward envs
Pretrain → SFT → RL

Nemotron 3 Nano

Efficient MoE hybrid Mamba-Transformer optimized for edge and PC deployment with 3.3× higher inference throughput.

31.6B total / 3.6B active · 25T tokens · 1M ctx
Pretrain → SFT → RL

Nemotron 3 Nano Omni

Native text, image, video, and audio support — a multimodal perception sub-agent for agentic AI.

30B total / 3B active · multimodal
SFT → RL (MPO / text / vision) → Eval

Quickstart

Clone the repo and explore the step catalog.

# Clone the repository
git clone https://github.com/saceto/NMTRN3.git
cd NMTRN3

# Install with uv
uv sync

# List available pipeline steps
uv run nemotron steps list
